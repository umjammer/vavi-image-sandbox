# vavi-image-sandbox

Vavi Image API Sandbox

## formats

  * jpeg (simd) (WIP)
  * barcode
  * heif
  * webp

## filter

  * pixel operation

    * cropping
    * rotation
    * draw

  * quantizing

    * fixed color model
    * oct tree
    * Imagemagick
    * neural net

  * resampling

    * HexeReinLanczos
    * ZhomxLanczos

    * smilla
    * noids

  * [Imagemagick](https://github.com/umjammer/vavi-image-sandbox/wiki/ImageMagickFilter)
  * flood filling
