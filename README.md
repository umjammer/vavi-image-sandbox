# Vavi Image API #

## formats ##
  * artmaster 88 (old japanese pc pc-88,98 image format)
  * windows bitmap
  * gif (non lzw)
  * windows icon
  * ppm
  * qrcode
  * jpeg (simd)
  * svg
  * wmf
  * susie (windows)

## filter ##
  * blob detection
  * face detection
  * quantize
    * oct tree
    * image magik
    * neural net
  * blur
  * resample
    * ffmpeg
  * pseudo colorization ([android version](https://github.com/umjammer/vavi-apps-pseudocolorization))
